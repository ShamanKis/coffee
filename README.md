Coffee



!\[Скриншот](https://github.com/ShamanKis/coffee/blob/main/screenshot.png)



Проект был создан на заказ. Больше похож на шаблон, так как последующее заполнение нужной информации происходило после предоставления заказчику и содержало не публичную информацию.



Технологии:

* HTML5
* CSS3
* JavaScript



Как открыть:

Просто откройте файл 'index.html' в любом браузере.



Функционал:

* Адаптивная верстка
* Современный дизайн
* Простота и удобство пользования
* Гибкость настройки и редактирования



Цель проекта:

Проект создан для заказчик. А исходный вид сайта без подробной информации для портфолио.
